The Event Scheduler App

A simple OpenShift3 app developed using the full-stack JavaScript with MongoDB, Express, AngularJS, and Node.js.

What this app does: 1- Allows you to add new events as well as edit them. 2- Allows you to search for events by either event's title or date range. 3- Validates all events on the UI before they're persisted. The validation process will make sure that the required fields are filled in with valid data. 4- Allows you to remove events.
